# Project Title

Recipe Book in React Native

## Getting Started
### Prerequisites

* [NodeJS](https://nodejs.org/en/) - Install node.js

### Installing

1.Install latest version of expo
```
npm install -g expo-cli
```
2.Install dependencies
```
cd %project-folder
```
```
npm install
```
```
expo start --web
```
